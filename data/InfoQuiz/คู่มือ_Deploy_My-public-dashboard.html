คู่มือ Deploy My-public-dashboard (GitHub Pages)

เอกสารนี้สรุปขั้นตอนติดตั้งและตั้งค่า เพื่อใช้ทบทวนหรือติดตั้งใหม่ในอนาคต

1. สร้าง Repository

  • Repository: My-public-dashboard
  • Visibility: Public
  • Default Branch: main

2. โครงสร้างไฟล์

My-public-dashboard/
│
├── index.html
├── data/
│   ├── Biology/
│   ├── Chemistry/
│   ├── Hardware-Spec/
│   └── Software-Spec/
└── README.md

3. Upload

  1. เปลี่ยนชื่อไฟล์ Dashboard เป็น index.html
  2. อัปโหลด index.html และโฟลเดอร์ data/
  3. Commit changes

4. เปิด GitHub Pages

  • Settings → Pages
  • Deploy from a branch
  • Branch: main
  • Folder: / (root)
  • Save

5. URL

https://owlvyhouse-academy.github.io/My-public-dashboard/

6. ตั้งค่าใน Dashboard (ครั้งแรก)

ช่องค่า
GitHub Usernameowlvyhouse-academy
RepositoryMy-public-dashboard
GitHub Tokenเว้นว่าง (Public Repository)

7. เพิ่มไฟล์ใหม่

  1. เพิ่มไฟล์ HTML ลงใน data/
  2. Commit ไปยัง GitHub
  3. กลับมาที่ Dashboard
  4. กด Sync GitHub

8. ตรวจสอบเมื่อมีปัญหา

  • Repository เป็น Public
  • Branch = main
  • มีไฟล์ index.html ที่ Root
  • ไฟล์ทั้งหมดอยู่ใน data/
  • Username = owlvyhouse-academy
  • Repository = My-public-dashboard

Checklist

  • ☐ GitHub Pages เปิดใช้งานแล้ว
  • ☐ เปิด URL ได้
  • ☐ Dashboard Sync สำเร็จ
  • ☐ ไฟล์ใน data/ แสดงครบ
